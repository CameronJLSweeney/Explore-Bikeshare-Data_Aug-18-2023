Question 2¶

What is the most common trip from start to end?¶

The code below finds the stations with the heaviest use across all cities.¶

In [24]:
top_stations <- c()

for (city in unique(dfs$City)) {
    city_data <- subset(dfs, City == city)
    station_counts <- table(city_data$Start.Station)
    max_count <- max(station_counts)
    top_stations_city <- names(station_counts[station_counts == max_count])
    top_stations <- c(top_stations, top_stations_city)
}
print(top_stations)
